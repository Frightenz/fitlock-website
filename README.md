# FitLock Website

Static website for FitLock, designed to run directly on GitHub Pages.

## Pages

- `index.html` - Marketing/home page
- `features.html` - App feature overview
- `support.html` - Public support URL and contact
- `privacy.html` - Public privacy policy
- `terms.html` - Terms of Use / EULA
- `404.html` - GitHub Pages fallback page

## GitHub Pages

Push these files to the repository branch you want to publish, then enable GitHub Pages in the repository settings.

Use these App Store Connect URLs after GitHub Pages is live:

- Privacy Policy URL: `https://<username>.github.io/<repo>/privacy.html`
- Support URL: `https://<username>.github.io/<repo>/support.html`
- Terms of Use URL: `https://<username>.github.io/<repo>/terms.html`
- Marketing URL: `https://<username>.github.io/<repo>/`
